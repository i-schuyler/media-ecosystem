The VPS initially had no usable Java, Gradle, Android SDK, ADB, FFmpeg, or
LAME installation. All downloaded tools and caches were placed in a
user-owned, task-specific local toolchain/cache area. No `sudo`, global
configuration change, unrelated SDK overwrite, or tracked local path was used.
Generated build output, Gradle state, `local.properties`, SDK components, and
debug signing state remain ignored.

The wrapper records the distribution SHA-256 and was generated by the verified
Gradle 9.5.0 distribution. The checked-in wrapper JAR hash is independently
verified during repository validation.

Android lint may report informational upgrade suggestions for API 37 or a
newer standalone Gradle release. Compile/target 36 and Gradle 9.5.0 are
intentional compatibility selections above, not hidden suppressions.
